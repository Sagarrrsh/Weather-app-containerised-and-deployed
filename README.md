# Smart Weather App — Containerized & Deployed on Kubernetes

A simple, modern weather application that:

✔ fetches real-time weather using OpenWeather API  
✔ auto-detects location via IP  
✔ provides AI-style suggestions (dress, umbrella, etc.)  
✔ is fully containerized using Docker  
✔ deployed on Kubernetes (NodePort / Argo CD ready)
